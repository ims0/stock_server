# 股票 K 线查看网站（Flask）

A 股 / 港股日线 K 线查看与交易模拟工具，基于 Flask + Plotly 构建。

---

## 快速启动

```bash
# 一键启动（自动创建虚拟环境、安装依赖、启动服务）
bash run.sh
```

或手动启动：

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
export TUSHARE_TOKEN="your_token_here"   # 可选，用于 TuShare 数据源
python3 app.py
```

浏览器打开：`http://127.0.0.1:5000`

## 主要功能

### K 线查看
- 支持 **A 股**（6 位数字代码，如 `600519`）和**港股**（1–5 位数字，如 `700` / `00700`）
- 支持按**中文名称**搜索股票（如输入 `腾讯`）
- **快速时间段**：下拉选择近 3/6/9 个月、1/1.5/2 年；也可手动指定起止日期
- 记忆上次查询的股票代码，下次打开自动填入
- 交互式 K 线图（Plotly），支持缩放、平移、底部 rangeslider 拖动
- **K 线样式**：阳线空心（红框白底）、阴线实心绿，柱体细线风格

### 均线
- 显示 **MA5**（默认开启）和 **MA10**（默认关闭）均线
- 图表上方提供开关，可随时切换显示/隐藏
- 均线计算自动跳过休市日（null 值），保证数值准确

### 交易模拟
- 图表上叠加**买入线**（绿色虚线）和**卖出线**（红色虚线）
- 两条线支持**上下拖动**（精度 0.1），或直接在输入框修改价格
- **成交逻辑**（贴近实盘）：
  - 买入：当日最低价 ≤ 买入限价时触发，成交价取 `min(买入价, 当日最高价)`
  - 卖出：当日最高价 ≥ 卖出限价时触发，成交价取 `max(卖出价, 当日最低价)`
- 交汇点在 K 线上标注**五角星**（买入绿色、卖出红色）
- 实时显示：**买入次数、卖出次数、资金倍数**（初始资金 100，全仓操作，不计手续费）
- **最优策略**按钮：自动枚举历史价格区间，一键找出当前时间段利润最大的买卖价位组合

### 缓存管理
- 历史 K 线数据本地 SQLite 缓存（`data/stock_cache.db`），重复查询无需再次请求网络
- 缓存同时记录休市日期（周末 + 节假日），避免重复补充
- 独立页面 `/cache` 可查看所有缓存条目及覆盖区间，支持按股票或全量清除缓存

## 数据源

按优先级自动切换，也可在页面手动指定：

| 数据源 | 说明 |
|--------|------|
| **AkShare** | 首选，免费，A 股 / 港股均支持 |
| **新浪财经** | 免费备用 |
| **TuShare** | 需配置 `TUSHARE_TOKEN` 环境变量 |
| **Yahoo Finance** | 免费，依赖网络可达性 |
| **Alpha Vantage** | 需配置 `ALPHAVANTAGE_API_KEY` 环境变量 |

---

## 环境变量

| 变量 | 说明 |
|------|------|
| `TUSHARE_TOKEN` | TuShare Pro Token（可选） |
| `ALPHAVANTAGE_API_KEY` | Alpha Vantage API Key（可选） |
| `PORT` | 服务监听端口，默认 `5000` |

---

## 技术栈

- **后端**：Python 3.10+、Flask 3、akshare、pandas、tushare、yfinance
- **前端**：原生 HTML/CSS/JS + Plotly.js（CDN）
- **存储**：SQLite（本地文件，无需额外部署）
